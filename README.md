El tiempo de duraci ́on del proyecto es de 12 meses, desde junio de 2015 a mayo de 2016. Se me-

en los archivos tisulares del IP.

analizar ́an estos niveles de expresi ́on, evaluando su rol en la alteraci ́on del ciclo celular de las c ́elulas
